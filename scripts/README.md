# Security Operations Labs

## Overview
This repository contains hands-on security operations (SOC) labs designed to demonstrate practical cybersecurity analyst skills in real-world enterprise environments. The focus is on alert triage, incident response, detection logic, vulnerability management workflows, and security operations documentation.

The content reflects day-to-day blue team responsibilities including investigating security events, prioritizing risk, documenting incidents, and continuously improving detection and response processes.
